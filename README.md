# TATA-top-car-sales-Dashboard
1. The Dashboard potreys Tata's top car models sale among the year 2024 & 2025
Car models are Nexon,Harieer,Punch,Altroz,Tiago
The Dashboard properly potreys each years individual models' sale, growth % of 2025's sale compared to 2024, actual unit sold each year, price of each model.

2. PURPOSE -
The Dashboard designed to help the users in tracking repots of car sales, have a insight of sales growth compared to last year, there is also have a report of top model sold with their price, which helps the user/company to find out theier customers budgets & choices.

3. TECH STACK -
The key technologies & functions are used in the dshboard are
EXCEL - Main Data table created in Excel
POWER BI - Main Data visualization platform used for report creation
POWER QUERY - Data transformation & cleaning layer for reshaping and preparing the Data.
